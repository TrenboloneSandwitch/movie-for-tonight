# Movie for tonight

- App for Choosing a Movie for the Evening Screening
- **Tech Stack**: Typescript, Remix, Prisma, Tailwind, Vitest, Zod, Conform

## Development

```sh
npx prisma init
yarn && yarn dev
```

## Deployment

First, build your app for production:

```sh
yarn build
```

Then run the app in production mode:

```sh
yarn start
```
